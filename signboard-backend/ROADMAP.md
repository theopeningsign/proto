## 🛠️ 기술 스택

### **Core 기술**
- **Phase 1**: 기존 CG 렌더링 시스템 (완료)
- **Phase 2**: Stable Diffusion WebUI + ControlNet
- **AI 제안**: OpenAI GPT-4o-mini + DALL-E 3

### **개발 환경**
- **OS**: Windows 11
- **GPU**: NVIDIA GTX 1050 Ti (4GB VRAM)
- **Python**: 3.10+
- **주요 라이브러리**: opencv, PIL, requests, openai

### **최적화 설정**
- **SD WebUI**: `--medvram --opt-split-attention --no-half-vae`
- **메모리 관리**: 512x512 이미지, batch_size=1
- **모델**: SD 1.5 + ControlNet v1.1

## 🎨 서비스 플로우

### **모드 1: AI 추천 모드** (신규)
```
사용자 입력: "30평 이탈리안 레스토랑"
     ↓
GPT 분석: 업종, 규모, 타겟 고객 파악
     ↓
상호명 제안: "라비타", "카사밀라", "트라토리아" (각각 이유 포함)
     ↓
로고 생성: 선택된 상호명에 맞는 로고 3개 생성
     ↓
간판 스타일 추천: 업종에 맞는 타입, 색상 추천
     ↓
사용자 선택: 상호명 + 로고 + 스타일 최종 선택
     ↓
Phase 1: CG 간판 생성
     ↓
Phase 2: SD 실사화
     ↓
결과: 완성된 실사 간판 (총 소요 시간: 2분)
```

### **모드 2: 직접 입력 모드** (기존)
```
사용자 입력: "카페 봄날" + 스타일 선택
     ↓
Phase 1: CG 생성
     ↓
Phase 2: SD 실사화
     ↓
결과: 완성된 실사 간판 (총 소요 시간: 30초)
```
